# Sobre o projeto

API REST desenvolvida com Spring Boot e MongoDB, simulando um sistema de usuários, posts e comentários.

Este projeto foi desenvolvido durante o curso de Java com Spring Boot da plataforma DevSuperior, ministrado pelo professor Nelio Alves, com o objetivo de consolidar os fundamentos de desenvolvimento backend com Java.

# Como executar o projeto

## Back end
Pré-requisitos: 
Java 25
Maven
Uma IDE (IntelliJ, Eclipse ou VS Code)
springbott
mongoDB

```bash
# clonar repositóriogit
git clone hhttps://github.com/Feliperossss/workshop-springboot4-mongodb.git

# Acessar a pasta do projeto

cd cd workshopmongo
./mvnw spring-boot:run

# Executar a aplicação

mvn spring-boot:run

# A aplicação ficará disponível em

http://localhost:8080
```
